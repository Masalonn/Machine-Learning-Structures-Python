MACHINE LEARNING FROM SCRATCH (PYTHON)
Implementation of neural network models and optimization algorithms in pure Python without high-level frameworks.

PROJECT OVERVIEW
A collection of core machine learning structures implemented from the ground up to demonstrate a deep understanding of underlying mathematical principles and algorithmic logic.

KEY COMPONENTS
Multilayer Perceptron (MLP): A custom neural network implementation solving the non-linear XOR problem using backpropagation.

Self-Organizing Maps (SOM): Implementation of Kohonen maps for unsupervised learning and data visualization.

Genetic Algorithms: A robust implementation of evolutionary computation used for optimization tasks.

TECHNICAL STACK
Language: Python 3

Libraries: NumPy (matrix operations), Matplotlib (data visualization)

Architecture: Modular design where each script functions as a standalone application.
